# ✋ AI Gesture Recognition Web App

A Streamlit-based web application that detects hand gestures in real time using MediaPipe and OpenCV.

## 🚀 Features
- Real-time hand tracking
- 21 hand landmarks detection
- Gesture recognition
- Open palm detection
- Scroll gesture detection
- Pause and resume gesture detection
